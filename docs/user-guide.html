MajorManager Documentation

User Guide

MajorManager is designed to easily manage your students, including emails and majors for quick reference. This guide will inform you of various functions in the program so that you can use them effectively.

Interface

Left Side

Clicking on the blank user icon allows you to upload a new profile picture.

Table

You can click on the header of any column to sort in ascending or descending order. The default is ascending by ID.

Right Side

The info boxes are, (from top to bottom), First Name, Last Name, Department, Major, Email, and optionally, a URL to a profile image. Major is a drop-down.
Filling them out will allow you to click Add and insert this person into the table.
You can quickly clear these fields with the Clear button.

If you click on a user in the table, their information will automatically populate in the fields. If you change this information and click Edit, it will update the table.

If you click Delete, they will be removed from the table. YOU CANNOT UNDO THIS, SO BE CAREFUL!

Menus

File

New Entry will take your currently filled in information and insert it into the table. If it is not filled out this option will be disabled.

Change Profile Picture will allow you to upload a new profile image.

Log Out will return you to the log in screen.

Exit Program will... well, take a guess. ;)

Edit

Edit, Delete, and Clear function the same as the buttons in the interface.

Theme

You can choose between a light and dark theme at any time by pressing the corresponding menu button.

Data

Import CSV will take data from a CSV file and push it into the table. Please check the CSV formatting guide before using this option.

Export CSV will export the data to a CSV file.

Formatting help will bring you to the aforementioned CSV guide.
