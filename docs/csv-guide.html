How to make and import CSV files.

A CSV file should look like this.

It should also be comma separated, without tabs, and newlines for every row. The output file should look like this. It is the default export in Excel. Note that the CSVTEST values are placeholders- replace them with your first names, last names, etc.

firstname,lastname,department,major,email
CSVTEST_F,CSVTEST_L,CSVTEST_D,CSVTEST_M,CSVTEST_E
CSVTEST2_F,CSVTEST2_L,CSVTEST2_D,CSVTEST2_M,CSVTEST2_E

Truthfully, it does not matter what the first row of your file is. It will always be ignored by MajorManager. But it is good practice to label them for your future reference.
